# v1 diagnosis

## Category rating

Event creation flow - 3.5/5 - Really easy to manage, but could use a list of what presents, and decorations.

Theme selection and visuals - 3/5 - Really basic. But could use like a few special effects when going on the event. Such as changing the background of the website sad with doves and rain when a funeral is being planned.

Guest list management - 4/5 - I like the part where you can put down their contact info in the guest list. But I think you could add a place box on when are they coming.

Activity planner - 4/5 - Like how it is designed and set up. But could use a list of what things are needed for the event.

Overall design - 3.5/5 - Simple but effective. Could use some fun effects upon being created. Such as when finished setting up a birthday event, confetti will appear.

Would someone actually use this - 3/5 - I think this is effective but the design needs to be more custom. And the simulation needs to be more 3D, realistic, and 3D.

## 3 strengths
1. The simplicity of the system.
2. How organized it is.
3. The notes, timelines, and simulation.

## 3 weaknesses
1. Not enough custom design and a bit too bland.
2. How insecure it is with the events.
3. The simulation sucks so far.

## 3 missing features
1. Login box.
2. The simulation animations.
3. Natural effects.

## Target user
Who would use this app - Parents and teenagers who want to plan out parties and events.

When will the audience use this - When a special event rolls by and they want to celebrate it.

What do they care about most when planning out an event - Location, guests, time, theme, and how is it going to turn out.

User statement - We are redesigning this for more young adults who want to plan out events for special days.

Redesign direction statement - Our AI version works, but it is too bland. We want to redesign it so that it feels more effective for young adults.
